Method 2: Check SQL Editor

  1. In Supabase Dashboard, click "SQL Editor"
  2. Run this query:
SELECT table_name
FROM information_schema.tables
WHERE table_schema = 'public'
AND table_type = 'BASE TABLE'
ORDER BY table_name;

This will show all your table names.

Method 3: Common Table Names to Try

If you just created your Supabase project, try these common names:

  • profiles - User profiles
  • todos - Todo items
  • messages - Messages/chat
  • items - Generic items
  • posts - Blog posts

Method 4: Create a Test Table

If you don't have any tables yet, create one in Supabase:

  1. Go to Table Editor
  2. Click "Create a new table"
  3. Name it: test_table
  4. Add some columns (id, name, created_at)
  5. Save and insert some test data
  6. Then access it at: http://localhost:8000/data/test_table/
